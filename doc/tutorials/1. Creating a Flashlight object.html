1. Creating a Flashlight instance.

This tutorial shows how to create a Flashlight instance. There are two ways:

1. Using The FlashlightAPI23

The constructor requires two parameters: CameraManager and cameraId(String).

Obtaining a CameraManager instance from the system service:

//in activity class
var cameraManager = getSystemService(Context.CAMERA_SERVICE) as CameraManager

Select the cameraId value from the getCameraIdList method. "0" is recommended:

var cameraId = cameraManager.getCameraIdList[0]

Initialization:

var flashlight = FlashlightAPI23(cameraManager, cameraId)

All code:

//in activity class
var cameraManager = getSystemService(Context.CAMERA_SERVICE) as CameraManager
var cameraId = cameraManager.getCameraIdList[0]
var flashlight = FlashlightAPI23(cameraManager, cameraId)

2. Using The FlashlightAPI23Builder

Builder initialization. The constructor's parameter is Context:

//in activity class
var flashlightBuilder = FlashlightAPI23Builder(this.applicationContext)

(Optional) Set the cameraId value using the setCameraId(String) method:

flashlightBuilder.setCameraId("0")

If you don't call this method, the cameraId value will be "0".

Initialization:

var flashlight = flashlightBuilder.getFlashlight()

All code:

//in activity class
var flashlightBuilder = FlashlightAPI23Builder(this.applicationContext)
var flashlight = flashlightBuilder.getFlashlight()
