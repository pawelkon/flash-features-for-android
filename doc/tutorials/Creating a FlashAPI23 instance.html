Creating a FlashAPI23 instance

This tutorial shows how to create a FlashAPI23 instance:

The constructor requires two parameters: CameraManager and cameraId(String).

Obtaining a CameraManager instance from the System service:

//in activity class
var cameraManager = applicationContext.getSystemService(Context.CAMERA_SERVICE) as CameraManager

Initialization:

var flash = FlashAPI23(cameraManager)

All code:
